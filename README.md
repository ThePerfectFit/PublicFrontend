# Setting Up React Development Environment

This guide will help you download Visual Studio Code (VS Code) and set up a React development environment on your computer.

## Prerequisites

Before you begin, ensure that you have the following installed on your computer:

1. **Node.js**: React requires Node.js to run. You can download it from the official website.
2. **npm**: npm (Node Package Manager) comes bundled with Node.js. It allows you to install libraries and tools.

## Step 1: Download Visual Studio Code

1. Go to the [Visual Studio Code website](https://code.visualstudio.com/).
2. Click on the **Download** button for your operating system (Windows, macOS, or Linux).
3. Follow the installation instructions provided on the website.

### For Windows Users:

- Download the installer and run it.
- Follow the prompts in the installation wizard.

### For macOS Users:

- Download the `.dmg` file and open it.
- Drag and drop Visual Studio Code into your Applications folder.

### For Linux Users:

- You can download the `.deb` or `.rpm` package based on your distribution.
- Follow the installation instructions specific to your distribution.

## Step 2: Install Node.js and npm

1. Visit the [Node.js website](https://nodejs.org/).
2. Download the LTS version for your operating system.
3. Run the installer and follow the instructions.

### Step 3: Set up Git
**Git** - Install Git by downloading it from [git-scm.com](https://git-scm.com/).

### Verify Installation

To confirm that Node.js and npm are installed correctly, open your terminal or command prompt and run the following commands:

```bash
node -v
npm -v

```
